# Calculator Design Pattern - Project Skeleton

This repository contains a minimal Python package `app` implementing a simple calculator and a small test suite.

Structure

- `app/` - package source
- `tests/` - pytest test suite

Quick start

1. Create a virtual environment and install test deps:

   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt

2. Run tests:

   pytest -q

Notes

This is a starter skeleton created to match the requested project layout. Implementations are intentionally small and easy to extend.
